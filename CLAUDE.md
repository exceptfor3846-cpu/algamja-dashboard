# Claude Code 작업 규칙

## 자율 실행 원칙
- 파일 생성, 폴더 생성, 패키지 설치, 명령어 실행 모두 허락 없이 자율적으로 진행
- 오류 발생 시 스스로 디버깅하고 수정하여 계속 진행
- 중간에 나에게 질문하거나 허락 구하지 말 것
- 모든 작업이 완벽히 완료되고 실제 실행까지 확인된 후에만 결과 보고

## 명령어 규칙 (중요)
- Python 실행: py 명령어 사용 (python 아님)
- 패키지 설치: pip install 사용 (가상환경 활성화 후)

## 가상환경 규칙 (중요)
- 모든 작업은 반드시 가상환경(venv) 안에서 진행
- 가상환경 없으면 먼저 생성: py -m venv venv
- 작업 전 항상 가상환경 활성화: venv\Scripts\activate
- 패키지 설치는 가상환경 활성화 후 pip install 사용
- requirements.txt 항상 최신으로 유지
